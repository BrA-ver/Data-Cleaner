### Shaders
Shaders used in this project include:
- A distortion shader to create a water effect
- A Hologram shader used for barriers and doors
- An outline shader used on the player

The shader graphs and materials can be found in Materials -> Shaders

### Controls 
This project uses the New Input System.

Mouse & Keyboard:
- WASD - Move
- Space - Jump
- Left Click - Throw Ball

Controller:
- Left Stick - Move
- South Button - Jump
- West Button - Throw Ball
